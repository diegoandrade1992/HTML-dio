História da Web

(voltar)

A web teve origem no final da década de 1960 e início da década de 1970, quando os primeiros computadores começaram a ser conectados em uma rede conhecida como ARPANET, desenvolvida pelo Departamento de Defesa dos Estados Unidos.
Esta rede inicialmente tinha fins militares e de pesquisa.

Em 1989, Tim Berners-Lee, um cientista da computação do CERN, propôs um sistema de gerenciamento de informações baseado em hipertexto para facilitar o compartilhamento de documentos entre pesquisadores. Esse sistema foi o embrião da World Wide Web.

Em 1990, Berners-Lee desenvolveu o primeiro navegador web, chamado de WorldWideWeb (mais tarde renomeado para Nexus), e o primeiro servidor web. Ele também criou as bases do HTML, URI e HTTP.

No início dos anos 1990, surgiram os primeiros navegadores populares, como o Mosaic, desenvolvido por Marc Andreessen e sua equipe na NCSA. O Mosaic foi o primeiro navegador a popularizar a web, tornando-a acessível para um público mais amplo.

Ao longo dos anos, a web continuou a se desenvolver com a criação de novas tecnologias, como CSS para controle de estilo, JavaScript para interatividade e muitos outros padrões e tecnologias que contribuíram para a web que conhecemos hoje.

Grandes nomes associados ao desenvolvimento da web incluem Tim Berners-Lee, Marc Andreessen, Sir Tim Berners-Lee, Vint Cerf, Robert Cailliau, entre outros. Cada um contribuiu de maneira significativa para a evolução e popularização da web.

O que são Clients

(voltar)

Clientes no contexto da web são os dispositivos ou softwares que acessam e consomem os recursos disponibilizados pelos servidores. Isso pode incluir navegadores da web (como Google Chrome, Mozilla Firefox, etc.), aplicativos web (como aplicativos de e-mail baseados na web), dispositivos móveis (como smartphones e tablets) e qualquer outro software que faça solicitações para os servidores da web.

o que são Servers

(voltar)

Os servidores na web são computadores ou sistemas de computadores que armazenam e fornecem recursos, dados e serviços para os clientes. Eles são responsáveis por processar as solicitações dos clientes e enviar as respostas apropriadas. Isso pode incluir servidores de hardware físico, bem como servidores de software, como servidores web (como Apache, Nginx), servidores de banco de dados (como MySQL, PostgreSQL), servidores de e-mail, entre outros. A hospedagem de sites é o serviço de fornecer espaço em servidores para hospedar sites e torná-los acessíveis na internet.

Linguagens de Programação

(voltar)

Uma linguagem de programação é um conjunto de instruções que permite a um programador escrever código para controlar o comportamento de um computador ou software. Existem duas categorias principais de linguagens de programação na web: server-side (lado do servidor) e client-side (lado do cliente).

Server-side:

São linguagens de programação que são executadas nos servidores para gerar conteúdo dinâmico e processar solicitações dos clientes. Exemplos incluem: PHP, Python, Ruby, Java, C#, entre outros. Essas linguagens são usadas para criar a lógica de negócios, acessar bancos de dados e gerar páginas da web dinâmicas.

Client-side:

São linguagens de programação que são executadas nos navegadores dos clientes para manipular a interface do usuário e interagir com o conteúdo web. Exemplos incluem JavaScript, HTML (embora tecnicamente não seja uma linguagem de programação, é uma linguagem de marcação), CSS. O JavaScript é a linguagem mais comum para programação do lado do cliente e é usado para adicionar interatividade e dinamismo às páginas da web.

Estrutura Básica do HTML

(voltar)

O HTML segue uma estrutura básica composta por elementos principais, como <html>, <head> e <body>. O elemento <html> envolve todo o conteúdo da página, <head> contém metadados como título e links para estilos e scripts, enquanto <body> contém o conteúdo visível da página.

Tags e Atributos das Tags

(voltar)

As tags são elementos usados para marcar diferentes partes do conteúdo da página. Cada tag pode ter atributos que fornecem informações adicionais sobre o elemento. Por exemplo, a tag <img> é usada para exibir imagens e pode ter atributos como src para especificar a localização da imagem e alt para fornecer um texto alternativo.

Textos

(voltar)

Textos podem ser adicionados à página usando tags como <p> para parágrafos, <h1> a <h6> para títulos e subtítulos (sendo <h1> o mais importante e <h6> o menos), e para marcar partes específicas de um texto.

Listas Ordenadas e Não Ordenadas

(voltar)

Listas podem ser representadas usando <ul> para listas não ordenadas e <ol> para listas ordenadas. Cada item da lista é representado por <li>, dentro das tags <ul> ou <ol>.
